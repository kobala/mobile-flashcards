# Mobile Flashcards 

## About Project

React Native + Redux flashcards app for iOS and Android

## Installing

````
$ git clone https://github.com/kobala/mobile-flashcards.git
$ cd mobile-flashcards
$ yarn install (or npm install)
$ yarn start (or npm start)
````
